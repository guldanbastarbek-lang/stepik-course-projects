# stepik-course-projects
# Python бағдарламалау тілінде цифрлық өнімдер мен жобаларды жасау әдістемесі

Бұл репозиторий болашақ информатика мұғалімдерінің жобалық құзыреттілігін қалыптастыруға бағытталған педагогикалық эксперименттің практикалық материалдарын жинақтау орны болып табылады.

## 🎯 Эксперимент мақсаты
Stepik платформасындағы теориялық және аралық диагностикалық білімді (№287981 курс) GitHub кәсіби әзірлеу ортасымен интеграциялау арқылы студенттердің дербес цифрлық өнімдер (Telegram-боттар, деректерді талдау жобалары) жасау дағдыларын дамыту.

## 📊 Модель құрылымы: Stepik + GitHub
* **Stepik (Аралық диагностика):** Синтаксисті автоматты тексеру, теориялық модульдер, базалық алгоритмдер.
* **GitHub (Кәсіби орта):** Дайын цифрлық өнімді орналастыру, нұсқаларды басқару (commits), жобаны техникалық құжаттандыру (README.md).

## 📅 15 апталық курс шеңберіндегі жобалық тапсырмалар
* **1-7 апта:** Базалық синтаксис пен скрипттер (Stepik платформасында орындалады).
* **8-11 апта:** Мәліметтермен жұмыс және Pandas кітапханасы (Прототиптерді GitHub-қа жүктеу).
* **12-15 апта:** Толыққанды цифрлық өнімдер — Функционалды Telegram-боттар мен интерактивті жобалар (GitHub ортасында қорғалады).
